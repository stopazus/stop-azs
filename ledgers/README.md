# Ledgers Directory

This directory contains financial ledgers and transaction records.

## Subdirectories

### `active/`
Current working Excel/CSV workbooks for ongoing analysis.

- Live transaction records
- Working spreadsheets
- Files under active modification

### `snapshots/`
Weekly frozen CSV exports for historical reference and audit trail.

Files are named with date stamps (e.g., `ledger_2025-11-22.csv`) to:
- Preserve point-in-time records
- Create audit trail
- Enable historical comparison

**Naming Convention**: `ledger_YYYY-MM-DD.csv`
