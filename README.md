# stop-azs

This repository provides reusable templates to support financial investigation casework.

## Included templates
- `docs/investigation_template.md` – Narrative report outline to capture case context, transaction review findings, and recommended actions.
- `docs/ledger_analysis_template.csv` – Ledger analysis worksheet for recording transaction-level attributes, flags, and follow-up ownership.

## Getting started
1. Duplicate the templates into your working folder.
2. Update headings and checklists to reflect the specifics of your investigation.
3. Store any sensitive data outside of this repository and follow your organization's handling requirements.
