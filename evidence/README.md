# Evidence Directory

This directory contains all evidence files related to the case, organized for integrity and traceability.

## Subdirectories

### `raw/`
Original PDFs and images including:
- Bank statements
- Email correspondence
- Original source documents

**Important**: Files in this directory should be preserved in their original form. Do not modify or rename files here.

### `hashed/`
Renamed copies of evidence files using SHA-256 hash for integrity verification.

Files are named using their SHA-256 hash to:
- Ensure file integrity
- Prevent tampering
- Create verifiable audit trail

### `ucc_filings/`
Official UCC-1 filings and registry snapshots.

Contains:
- UCC-1 financing statements
- Registry snapshots
- Official filing documentation
